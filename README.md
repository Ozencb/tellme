# Tellme
A CLI for Chat-GPT, written in Go

## Usage

```
tm <prompt> [options]
```

```
Usage:
  tm [flags]

Flags:
  -h, --help           help for tm
  -m, --model string   GPT Model. Default: gpt-3.5-turbo (default "gpt-3.5-turbo")
  -r, --reset          Reset API Key
```
